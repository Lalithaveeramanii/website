Php

PHP is a server-side scripting language used to build dynamic web pages and applications by processing data and interacting with databases.

C Programming

C is a powerful, general-purpose programming language that provides low-level access to memory.It forms the foundation for many modern languages like C++ and Java.

C++

C++ is an extension of C that adds object-oriented programming features, enabling more structured and reusable code.

Java

Java is a versatile, platform-independent programming language known for its "write once, run anywhere" capability.

Python

Python is a high-level, versatile programming language known for its readability and simplicity, making it ideal for beginners and rapid development.

MongoDB/MySql

MongoDB is a NoSQL database that stores data in flexible, JSON-like documents.MySQL is a relational database management system that uses structured query language.

C# .NET

C# is an object-oriented programming language developed by Microsoft for building a variety of applications.NET is a framework that provides tools and libraries for C# development, enabling cross-platform application creation and deployment.
